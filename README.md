# Portfolio Website

My personal portfolio website showcasing my projects and skills.

## Features
- Responsive Design
- Contact Form
- Project Showcase
- Skills Section

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Node.js
- PostgreSQL

## Setup
1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables
4. Run the server: `node server.js`
